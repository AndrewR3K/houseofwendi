# House of Wendi Link Hub

Beautiful, single-page link tree for House of Wendi, built with Vue 3 + Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Update Social Links

Edit the `links` array in [src/App.vue](src/App.vue) to add the real Instagram, Whatnot, and Poshmark URLs.

## Logo

Replace the placeholder SVG at [src/assets/houseofwendi-logo.svg](src/assets/houseofwendi-logo.svg) with the official logo if desired.

## GitHub Pages

The base path is set for the repo name `/houseofwendi/` in [vite.config.ts](vite.config.ts). If the repo name changes, update the `base` value to match. Build the project and publish the `dist` folder to GitHub Pages using your preferred method.
